## 📌 Project Overview  
This project simulates a full **Risk‑Based Quality Management (RBQM)** cycle for a fictional Phase II clinical study titled:

**GlucoBalance‑50: Evaluation of a Pseudo‑Natural Antidiabetic Agent**  
**N = 200 | Duration = 24 months**

The simulation includes:  
- Protocol‑risk assessment  
- Construction of study‑specific **KRIs and QTLs**  
- Centralized Monitoring dashboard  
- Weekly refresh and signal detection  
- Corrective action & escalation logging  
- Interpretation of signals  
- Specification‑like documentation
