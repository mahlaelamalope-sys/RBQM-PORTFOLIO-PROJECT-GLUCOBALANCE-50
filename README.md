## 📌 Project Overview  
This project simulates a full **Risk‑Based Quality Management (RBQM)** cycle for a fictional Phase II clinical study titled:

**GlucoBalance‑50: Evaluation of a Pseudo‑Natural Antidiabetic Agent**  
**N = 200 | Duration = 24 months**

The simulation includes:  
- Protocol‑risk assessment  
- Construction of study‑specific **KRIs and QTLs**  
- Centralized Monitoring dashboard  
- Weekly refresh and signal detection  
- Corrective action & escalation logging  
- Interpretation of signals  
- Specification‑like documentation

This portfolio demonstrates hands‑on capability aligned with IQVIA’s **Centralized Monitoring Analyst (CMA)** and **CluePoints Lead Programmer** expectations.

## 📊 Key Outputs  
### 🔹 Risk Map  
Includes high‑impact risks for:  
- Informed consent deviations  
- AE under‑reporting  
- Visit window deviations  
- Data entry delays  
- Missing CRF pages  

### 🔹 KRIs & QTLs  
Includes 5 fully defined KRIs:  
- Protocol deviations  
- SAE timeliness  
- Missing CRF pages  
- Data entry timeliness  
- Visit window adherence  

Each with:
- Definition  
- Target  
- Study‑level QTL  
- Threshold logic  

### 🔹 Weekly Dashboard  
A CluePoints‑style view including:  
- Red/Yellow/Green signals  
- Interpretation comments  
- Site‑level issues  
- Trend patterns  

### 🔹 Action Log  
Tracks:  
- Risk  
- Corrective actions  
- Responsible owner  
- Deadlines
